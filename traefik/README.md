# Traefik

Traefik is a modern, cloud-native reverse proxy and ingress controller. It automatically discovers services and configures routing, supports HTTP, TCP, UDP, WebSockets, and gRPC, and provides automatic HTTPS via Let's Encrypt. It integrates natively with Docker, Kubernetes, and other orchestrators.

**Website:** [traefik.io](https://traefik.io/traefik/)
