# Vaultwarden

Vaultwarden is a lightweight, self-hosted password manager server compatible with all official Bitwarden clients. Written in Rust, it uses far fewer resources than the official Bitwarden server, making it ideal for personal or small-team deployments.

**Website:** [github.com/dani-garcia/vaultwarden](https://github.com/dani-garcia/vaultwarden)

## Notes

Vaultwarden is not affiliated with Bitwarden, Inc. It was formerly known as Bitwarden_RS.
