# Webtop

Webtop provides a full Linux desktop environment accessible through any modern web browser. Built by LinuxServer.io, it offers Alpine, Ubuntu, Fedora, and Arch-based containers with desktop environments including XFCE, KDE, MATE, and i3.

**Website:** [github.com/linuxserver/docker-webtop](https://github.com/linuxserver/docker-webtop)
