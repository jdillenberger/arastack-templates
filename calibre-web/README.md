# Calibre-Web

A web-based ebook management interface that uses an existing Calibre database to provide browsing, reading, and downloading of ebooks through a clean web UI. Supports user management, OPDS feeds for e-reader apps, metadata editing, and on-the-fly format conversion.

**Website:** [github.com/janeczku/calibre-web](https://github.com/janeczku/calibre-web)
