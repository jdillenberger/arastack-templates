# Obsidian LiveSync

This template deploys a CouchDB backend for the [Obsidian LiveSync](https://github.com/vrtmrz/obsidian-livesync) plugin, enabling real-time vault synchronization across devices. It also includes a web-based Obsidian interface.

**Website:** [obsidian.md](https://obsidian.md)

## Notes

- This is not Obsidian itself (a local app), but the server-side sync infrastructure. You still need the Obsidian app with the LiveSync plugin installed on each device.
- Incompatible with Obsidian's official Sync service.
