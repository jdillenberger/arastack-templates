# Stirling-PDF

Stirling-PDF is a powerful, open-source PDF editing platform with over 50 tools for merging, splitting, converting, signing, redacting, OCR, and compressing PDFs. It can run as a desktop app, in the browser, or self-hosted on your own server with a REST API for automation.

**Website:** [github.com/Stirling-Tools/Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF)
