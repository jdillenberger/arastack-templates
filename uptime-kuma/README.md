# Uptime Kuma

Uptime Kuma is a self-hosted monitoring tool with a clean, reactive UI. It supports monitoring via HTTP(s), TCP, Ping, DNS, WebSocket, and Docker containers, and can send alerts through 90+ notification services including Telegram, Discord, and Slack.

**Website:** [github.com/louislam/uptime-kuma](https://github.com/louislam/uptime-kuma)
